# Carpeta de imágenes

Coloca aquí el archivo del logo principal.

**Nombre del archivo:** `logo.png`

El logo debe ser una imagen en formato PNG, JPG o SVG que se mostrará en el header de la aplicación.

Si el logo tiene otro nombre o formato, actualiza la importación en `src/components/Header/Header.jsx`:

```javascript
import logoImage from '../../assets/images/tu-logo.png';
```
